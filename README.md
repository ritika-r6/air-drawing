# ✋ Air Drawing

A real-time hand gesture drawing application built with Python and OpenCV.

## What it does
- Tracks your fingertip through a webcam
- Lets you draw on a virtual canvas just by moving your hand in the air
- Uses color segmentation and contour detection to detect hand position

## Tech Used
- Python
- OpenCV

## How to Run
1. Install OpenCV: `pip install opencv-python`
2. Run: `python air_drawing.py`
